# 🌐 Exercícios Práticos: HTML5 & CSS3 - Uniube

Repositório dedicado ao armazenamento de atividades e exercícios práticos desenvolvidos durante a graduação na **Uniube (Universidade de Uberaba)**. O projeto foca no domínio das tecnologias fundamentais da Web.

---

## Objetivo do Projeto
Demonstrar a evolução no aprendizado de front-end, cobrindo desde a marcação semântica básica até a estilização avançada e organização de layouts.

## Tecnologias e Conceitos Aplicados
* **HTML5 Semântico:** Uso correto de tags como `<header>`, `<main>`, `<footer>`, `<table>` e `<form>`.
* **CSS3:** Manipulação de cores, tipografia, modelos de caixa (*Box Model*), posicionamento e seletores.
* **Formulários:** Validação de campos e estruturação de dados de entrada.
* **Layouts:** Práticas de estruturação com tabelas e divs.

## Estrutura do Repositório

O projeto está dividido em duas frentes principais de estudo:

### 1. PetersonHonorato (HTML5 & CSS3)
Esta pasta contém o alicerce do desenvolvimento web, focada na criação de interfaces estáticas e estilização.
* **fale_conosco.html**: Implementação de formulários e coleta de dados.
* **imagens.html**: Manipulação e otimização de elementos visuais.
* **links.html**: Estrutura de navegação e hiperlinks.
* **index.html**: Home page integrando os conceitos de marcação.
* **Conceitos Aplicados**: Semântica HTML, seletores CSS e Box Model.

### 2. PetersonHFreitas (Lógica & JavaScript)
Esta pasta foca na interatividade e no comportamento das páginas, utilizando scripts para tornar a experiência do usuário dinâmica.
* **exercicio1 a 4.html**: Atividades práticas de lógica de programação.
* **style.css**: Estilização dedicada para os componentes interativos.
* **index.html**: Ponto de entrada para os módulos de JavaScript.
* **Conceitos Aplicados**: Variáveis, estruturas condicionais, funções e manipulação do DOM.

---

## Tecnologias Utilizadas
* **HTML5**: Estruturação semântica.
* **CSS3**: Design, cores e layout.
* **JavaScript**: Lógica e interatividade.
